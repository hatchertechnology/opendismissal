Bulk Import Dismissal Codes

Upload a CSV file to bulk update dismissal codes for students. The CSV file must contain the following columns:

  • student_name - Exact student name (case-insensitive)
  • dismissal_code - New dismissal code (1-8 characters, letters and numbers only). Leave blank to auto-generate.

CSV Format Example:

student_name,dismissal_code
John Smith,ABC123
Jane Doe,XYZ789
Bob Johnson,

Important Notes:

  • Student names must match exactly (case-insensitive)
  • Dismissal codes must be 1-8 characters long
  • Only letters and numbers are allowed in codes
  • Codes must be unique across all students
  • Leave dismissal_code blank to auto-generate a 6-character code
  • All changes will be logged for audit purposes
