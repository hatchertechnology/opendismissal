OpenDismissal Real-time Sync Demo

This demo shows how real-time updates are broadcast to all connected browsers when dismissal events occur.

How to test:

  1. Click "Connect" to establish WebSocket connection
  2. Open another browser tab/window to this same page and connect
  3. Use the OpenDismissal mobile interface or API to log parent arrivals or complete pickups
  4. Watch real-time updates appear in both browser windows instantly

Events to watch for:

  • Parent Arrived: When a parent checks in using their dismissal code
  • Pickup Completed: When a student is released to their parent
  • Status Changed: When a student's status is manually updated
  • Dismissal Reset: When all students are reset to "Waiting" status
